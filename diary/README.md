## ✨ 기능 소개

### 일기 보관함
- **일기 작성**: 기분(행복해요, 슬퍼요, 놀랐어요, 화나요, 기타)을 선택하고 제목과 내용을 작성
- **일기 목록**: 카드 형태로 일기 목록을 표시 (페이지당 12개)
- **일기 상세 보기**: 작성한 일기를 상세히 확인
- **일기 수정/삭제**: 작성한 일기를 수정하거나 삭제
- **필터링**: 기분별로 일기 필터링
- **검색**: 제목으로 일기 검색
- **페이지네이션**: 일기 목록을 페이지 단위로 탐색

### 사진 보관함
- **강아지 사진 갤러리**: Dog CEO API를 활용한 랜덤 강아지 사진 표시
- **무한 스크롤**: 스크롤 시 자동으로 추가 사진 로드
- **사진 형태 변경**: 기본형(1:1), 가로형(4:3), 세로형(3:4) 선택 가능

### 상세 페이지
- **일기 상세 정보**: 제목, 내용, 날짜, 기분 표시
- **내용 복사**: 일기 내용을 클립보드에 복사
- **회고 기능**: 일기에 회고를 추가하고 날짜와 함께 저장
- **수정/삭제**: 일기 수정 및 삭제 기능

## 🛠 기술 스택

- **HTML5**: 마크업 구조
- **CSS3**: 스타일링 및 반응형 디자인
- **JavaScript (Vanilla)**: 모든 기능 구현
- **LocalStorage**: 브라우저 로컬 스토리지를 활용한 데이터 저장
- **Dog CEO API**: 강아지 사진 제공

## 🚀 시작하기

### 1. 프로젝트 클론 또는 다운로드

```bash
cd diary
```

### 2. 웹 서버 실행

로컬 웹 서버를 실행하거나 브라우저에서 직접 `index.html` 파일을 열 수 있습니다.

**Python을 사용한 간단한 서버 실행:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**Node.js를 사용한 서버 실행:**
```bash
npx http-server
```

### 3. 브라우저에서 접속

```
http://localhost:8000
```

### 4. 테스트 데이터 생성 (선택사항)

브라우저 콘솔에서 `seed_diary.js`를 실행하거나, 개발자 도구에서 다음 코드를 실행하여 테스트용 일기 데이터를 생성할 수 있습니다:

```javascript
// seed_diary.js 파일을 HTML에 포함시키거나 콘솔에서 실행
```

## 📝 주요 기능

### 일기 작성
1. "일기쓰기" 버튼 클릭
2. 기분 선택 (행복해요, 슬퍼요, 놀랐어요, 화나요, 기타)
3. 제목과 내용 입력
4. 등록 버튼 클릭 (모든 필드 입력 시 활성화)

### 일기 필터링
- 상단 필터 메뉴에서 기분 선택
- "전체" 선택 시 모든 일기 표시

### 일기 검색
- 검색 입력창에 제목 키워드 입력
- 2초 후 자동으로 검색 결과 표시

### 페이지네이션
- 하단 페이지 버튼으로 페이지 이동
- 이전/다음 버튼으로 페이지 범위 이동
- 페이지당 12개의 일기 표시

### 일기 상세 보기
- 일기 카드 클릭 시 상세 페이지로 이동
- 일기 내용 복사 기능
- 회고 추가 및 관리
- 일기 수정 및 삭제

### 사진 보관함
- "사진보관함" 탭 클릭
- 스크롤 시 자동으로 추가 사진 로드
- 필터에서 사진 형태 선택 (기본형/가로형/세로형)

일기 상세 정보를 표시하는 페이지입니다.

### `detail.js`
- 일기 상세 정보 표시
- 일기 수정/삭제 기능

## 💾 데이터 저장

모든 일기 데이터는 브라우저의 **LocalStorage**에 저장됩니다.

- 키: `일기카드들`
- 형식: JSON 배열
- 구조:
```javascript
{
  기분: "행복해요",
  제목: "일기 제목",
  내용: "일기 내용",
  날짜: "2024.01.01",
  회고록: [
    {
      회고내용: "회고 내용",
      회고날짜: "[2024.01.02]"
    }
  ]
}
```

## 📌 주의사항

- 브라우저의 LocalStorage를 사용하므로, 브라우저 데이터를 삭제하면 저장된 일기가 모두 삭제됩니다.
- Dog CEO API는 외부 서비스이므로 인터넷 연결이 필요합니다.
- `detial.html` 파일명에 오타가 있을 수 있습니다 (detail이 아닌 detial).
